# Nombre del Proyecto

Cinemax

## Descripción
Este proyecto es una aplicación web básica desarrollada en PHP, enfocada en practicar conceptos de desarrollo web y estructura de proyectos.

## Tecnologías utilizadas
- PHP
- HTML
- CSS
- JavaScript
- Servidor local (XAMPP)

## Requisitos
- Tener instalado XAMPP o cualquier servidor con soporte PHP
- Navegador web actualizado

## Cómo ejecutar el proyecto
1. Copiar la carpeta del proyecto dentro de la carpeta `htdocs` de XAMPP  
2. Iniciar Apache desde el panel de control de XAMPP
